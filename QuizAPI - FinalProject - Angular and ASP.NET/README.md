# QuizAPI - Final Project (Angular & ASP.NET)

A full-stack quiz application built with Angular frontend and ASP.NET Core backend. This project demonstrates RESTful API integration, user authentication, quiz management, and responsive UI design.

## 🛠️ Tech Stack

**Frontend:**
- Angular 17+
- TypeScript
- Bootstrap / Tailwind CSS
- RxJS for state management

**Backend:**
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server / SQLite
- JWT Authentication

**Tools:**
- Swagger UI for API documentation
- Git for version control

## 🚀 Quick Start

### Prerequisites
- Node.js (v18+)
- .NET SDK (v8+)
- Database (SQL Server or SQLite)

### Backend Setup
```bash
cd "QuizAPI - FinalProject - Angular and ASP.NET/Backend"
dotnet restore
dotnet ef database update
dotnet run
